# virtualboardgame


## Development

### Setup

#### Pre-requisites

Node.js:

Download from [https://nodejs.org/en/](https://nodejs.org/en/)

Yarn:

```
npm install -g yarn
```

### Install

Run npm install in the root of the repo.

```bash
cd virtualboardgames
npm install
```

### Start build

```bash
npm run build
```
